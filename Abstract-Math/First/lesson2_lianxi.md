# 扩展练习 F：集合运算（巩固版）

本练习涵盖并集、交集、差集、对称差，并附解析。

---

## 练习 F1（基础运算）

设：

A = {1,2,3,4},  B = {3,4,5,6}

请计算：

1. A ∪ B  
2. A ∩ B  
3. A \ B  
4. B \ A  
5. (A \ B) ∪ (B \ A)  

**参考答案：**

1. A ∪ B = {1,2,3,4,5,6}  
2. A ∩ B = {3,4}  
3. A \ B = {1,2}  
4. B \ A = {5,6}  
5. (A \ B) ∪ (B \ A) = {1,2,5,6}

---

## 练习 F2（差集方向练习）

设：

C = {a,b,c}, D = {b,c,d}

请计算：

1. C \ D  
2. D \ C  
3. (C \ D) ∪ (D \ C)

**参考答案：**

1. C \ D = {a}  
2. D \ C = {d}  
3. (C \ D) ∪ (D \ C) = {a,d}

---

## 练习 F3（对称差应用）

设：

X = {1,2,3,4}, Y = {2,4,6,8}

1. X △ Y = ? （对称差）  
2. 检查对称差是否满足交换律：X △ Y = Y △ X？

**参考答案：**

1. X △ Y = (X \ Y) ∪ (Y \ X) = {1,3} ∪ {6,8} = {1,3,6,8}  
2. Y △ X = {6,8} ∪ {1,3} = {1,3,6,8} ✅  
   → 对称差满足交换律

---

## 练习 F4（综合运算 + 思考题）

设：

P = {1,2,3,4}, Q = {3,4,5,6}, R = {2,4,6,8}

请计算：

1. (P ∪ Q) ∩ R  
2. P ∩ (Q ∪ R)  
3. (P \ Q) ∪ (Q \ R)  

**参考答案：**

1. P ∪ Q = {1,2,3,4,5,6}  
   (P ∪ Q) ∩ R = {2,4,6}  
2. Q ∪ R = {2,3,4,5,6,8}  
   P ∩ (Q ∪ R) = {2,3,4}  
3. P \ Q = {1,2}  
   Q \ R = {3,5}  
   (P \ Q) ∪ (Q \ R) = {1,2,3,5}

---

## 小提示与练习技巧

1. **差集方向一定要注意**  
   - A \ B ≠ B \ A  
   - 先列左集合元素，再逐个去掉右集合中元素

2. **交集和并集逻辑**
   - 并集 ∪ → OR（取所有元素）  
   - 交集 ∩ → AND（取共有元素）

3. **对称差**  
   - 只取只属于一个集合的元素  
   - 满足交换律 X △ Y = Y △ X

4. **Venn 图可视化**
   - 小集合画 Venn 图，直观理解运算结果  
   - 对复杂集合或三集合运算尤其有用

## 拓展练习（挑战题）

1. 证明：对任意集合 A，有 A \ A = ∅  
2. 证明：A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C)  
3. 证明：A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C)  
4. 画出三个集合 A, B, C 的 Venn 图，并标出 (A △ B) ∩ C 的区域  
5. 给出集合 X, Y, Z，使得 (X △ Y) △ Z ≠ X △ (Y △ Z)（检查对称差结合性）
