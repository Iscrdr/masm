# 第 1 课：集合与元素、表示法与空集（学术版）

## 一、核心概念

### 1. 集合（Set）

由一些对象构成的整体，用大写字母表示，如 A, B, X。

### 2. 元素（Element）

集合中的对象，用小写字母表示，如 a, b, x。

属于关系：\
- 若 a 属于 A，写作 **a ∈ A**\
- 若 a 不属于 A，写作 **a ∉ A**

### 3. 空集（Empty Set）

不包含任何元素的集合，记作 **∅** 或 `{}`。

------------------------------------------------------------------------

## 二、集合的表示法

### 1. 列举法（Roster）

直接写出所有元素，例如\
A = {1, 2, 3}

### 2. 描述法（Set-builder）

用性质描述：\
B = {x \| x 是偶数且 1 ≤ x ≤ 10}

### 3. 区间表示（针对实数）

\[0, 1\] = {x ∈ ℝ \| 0 ≤ x ≤ 1}

------------------------------------------------------------------------

## 三、基本性质小结

-   集合无序，无重复：{1,2} = {2,1}\
-   元素完全相同时集合相等\
-   空集是任何集合的子集（将在练习中思考）
