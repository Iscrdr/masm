# 第 2 课：集合运算（基础）

本节一次讲一点：并集、交集、差集。稍后会讲补集与笛卡尔积。

---

## 1. 并集（Union）

**定义：**

\[
x \in A \cup B \iff (x \in A) \text{ 或 } (x \in B)
\]

直观：把 A、B 两个集合的元素合并（去重）。

**例子：**

A = {1,2,3}, B = {3,4}  
A ∪ B = {1,2,3,4}

---

## 2. 交集（Intersection）

**定义：**

\[
x \in A \cap B \iff (x \in A) \text{ 且 } (x \in B)
\]

直观：A、B 共有的元素。

**例子：**

A ∩ B = {3}

---

## 3. 差集（Difference）

**定义：**

\[
x \in A \setminus B \iff (x \in A) \text{ 且 } (x \notin B)
\]

直观：在 A 中但不在 B 中的元素（A 减去 B）。

**例子：**

A \ B = {1,2}

---

## 小技巧（逻辑对应）
- 并集：OR（或）  
- 交集：AND（且）  
- 差集：AND NOT（且不）
